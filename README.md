# Gait-Analysis-Project

Analysis of online raw gait data to replicate the results from the paper by Weiss et al.

## Project Overview
This project aims to reproduce the findings of the cited paper by Weiss et al. using publicly available raw gait data. The goal is to obtain results as close as possible to those reported in the original study.

## How to Run the Code
To execute the project, run `Main_GreenDays.m` in MATLAB. This script coordinates all the remaining functions and performs the full analysis workflow.

## Data
The raw gait data used in this project is taken from https://physionet.org/content/ltmm/1.0.0.
